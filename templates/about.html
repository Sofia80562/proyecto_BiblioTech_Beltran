Acerca de BiblioTech Beltrán

Este proyecto es parte de la asignatura de DESARROLLO DE APLICACIONES WEB, desarrollado por María Beltrán.

El objetivo es crear una plataforma eficiente para la consulta y gestión de recursos bibliográficos de manera digital.

Misión del Proyecto

  • Organizar el catálogo de libros mediante rutas dinámicas.
  • Implementar una interfaz limpia usando plantillas Jinja2.
  • Facilitar el acceso a la información desde cualquier navegador.
